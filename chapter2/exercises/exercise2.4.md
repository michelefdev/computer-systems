# Exercise 2.4 — Hex Arithmetic

A) 0x503C + 0x08 = 0x5044

```
   0x503C
 + 0x0008
 --------
   0x5044
```

B) 0x503C − 0x0040 = 0x4FFC

```
   0x503C
 - 0x0040
 --------
   0x4FFC
```

C) 0x503C + 0x0064 = 0x50A0

```
   0x503C
 + 0x0064
 --------
   0x50A0
```

D) 0x50EA − 0x503C = 0x00AE

```
   0x50EA
 - 0x503C
 --------
   0x00AE
```
